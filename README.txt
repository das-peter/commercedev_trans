Drupal Commerce (Dev) Trans - Installation profile

This installation profile installs the standard Drupal installation profile 
and the modules needed to create a mutliligual shop.

Place this profile into the profiles/commercedev_trans directory of your site and make
sure all the dependencies are in a valid modules directory for the site.

DEPENDENCIES:
